    Inspetor de elementos

    Uma ferramenta chamada Inspetor de Elementos está disponível em navegadores da web como Google Chrome, Firefox e Safari, que permite visualizar e examinar o código HTML, CSS e JavaScript ao vivo de uma página da web. Ele é um componente crucial da caixa de ferramentas de um desenvolvedor web. Ao abrir o Inspetor de Elementos em um navegador, você pode examinar o layout e o conteúdo da página, identificar elementos específicos, alterar as propriedades relacionadas ao estilo, limpar o código JavaScript e muito mais. Ele fornece uma visão detalhada dos elementos individuais da página, como botões, links, imagens e fórmulas, para que você possa entender como eles estão sendo renderizados e interagir com o código subjacente.

    Com o seletor de elementos, você pode escolher um elemento específico da página clicando sobre ele no painel de visualização ou usando uma ferramenta de seleção. Isso destacará o código HTML correspondente no painel de fonte de código e exibirá todas as propriedades CSS que foram aplicadas a ele na barra lateral. Você pode alterar essas propriedades para ver como elas afetam a aparência e o layout do elemento em tempo real. Além disso, o Inspetor of Elementos fornece ferramentas de depuração, como a capacidade de especificar pontos de interrupção de código JavaScript e monitorar o fluxo de execução durante determinados eventos ou ações.

    A estrutura básica do HTML

    A estrutura básica do HTML é formada por vários elementos que fornecem a estrutura e formam o conteúdo de uma página Web.

    Analisando melhor:

    HTML: Este elemento é o contêiner raiz de todo o conteúdo da página.

    Head: O elemento "head" contém informações sobre a página, como o título da página, metadados, scripts e estilos CSS

    Title: O elemento "title" define o título da página, que é exibido na barra de título do navegador ou na guia da página.

    Body: O elemento "body" contém todo o conteúdo visível da página, como texto, imagens, links, elementos de formulário, vídeos, etc.

    >O que são tags?

    Em resumo, as tags na programação são componentes ou marcadores que especificam a estrutura, o comportamento e a aparência de vários elementos do programa, ajudando a organizar e formatar o código. O uso de tags pode ser pensado como instruções ou comandos que ajudam os programadores a definir o comportamento e a aparência de vários elementos de software.

    Exemplos de tags:

      "<h1></h1>" - Título de maior valor hierárquico
      "<h2></h2>"
      "<h3></h3>"
      "<h4></h4>"
      "<h5></h5>"
      "<h6></h6>" - Título de menor valor hierárquico

    Tags de texto

      "<p> </p>" – Principal tag de texto, compõe um parágrafo;
      "<span> </span>" – Apesar de ter uma funcionalidade e características parecidas com os parágrafos, costumam ser utilizadas apenas para pequenas informações, como legendas de um formulário, legendas de uma imagem, etc. Também pode ser utilizada para formar um container;
      "<pre> </pre>" – Tag utilizada para representar texto pré-formatado. Muito utilizada para inserir códigos;
      "<b> </b>" – Transforma o conteúdo em negrito;
      "<i> </i>" – Transforma o conteúdo em itálico;
      "<br/>" – Essa tag não necessita de fechamento, ela executa a função de quebra de linha.
      "<hr/>" – Essa tag não necessita de fechamento, ela forma uma linha horizontal.

    Atritutos de Tags

    Atributos são basicamente propriedades que uma tag pode conter, alguns atributos são mais globais e genéricos, o que significa que todas as tags terão. Também existem atributos que são mais específicos para cada tag.

    Exemplo de um atributo genérico: A tag "strong" pode conter a tag "id", que é nada mais nada menos que um identificador de um elemento que você queira alterar

    Exemplo de um atributo específico: A tag "input" proporciona um campo de texto no arquivo HTML, esta tag possui um atributo específico para ela, sendo ela o "type", se for colocado o valor "number" dentro da mesma, este campo de texto passará a ser exclusivo de números.

    Estes foram só alguns exemplos simples, mas resumindo, os atributos e propósitos específicos podem diferir dependendo da linguagem de programação ou biblioteca/estrutura que está sendo usada. Portanto, é crucial revisar a documentação pertinente para entender quais atributos estão disponíveis e como usá-los corretamente em uma determinada situação.

    Criação de Textos

    Textos são tipografias de HTML. Existem muitos elementos e atributos HTML que podem ser usados ​​para controlar o tipo de letra, incluindo font>, u>, sup> e sub>, entre outros. No entanto, é crucial observar que, em termos de práticas recomendadas, é recomendável usar CSS (Cascading Style Sheets) em vez de confiar apenas nos atributos HTML para controlar a aparência e o estilo da tipografia.     Listas ordenadas e não ordenadas HTML

    A tag HTML "ol" (lista ordenada) é usada para criar uma lista ordenada. Os itens da lista são definidos dentro da tag "ol" usando a tag "li" (item da lista). Aqui está um exemplo de como criar uma lista HTML ordenada:

    Exemplo de lista ordenada:

      <ol>
      <li>Item 1</li >
      <li>Item 2</li >
      <li>Item 3</li >
      </ol>

    Neste exemplo, temos uma lista com três itens organizados. Cada item é especificado entre as tags li>, que designam um item da lista. Quando renderizada em um navegador, a lista será numerada automaticamente.

    O item será exibido da seguinte forma no navegador:

    1. Item 1
    2. Item 2
    3. Item 3

    Lista não ordenada

    Um elemento de lista HTML é usado para agrupar e exibir uma lista de itens sem uma ordem claramente definida. Ao contrário de uma lista ordenada, onde os itens são mostrados em ordem numérica ou específica, uma lista que não é ordenada é exibida usando marcadores (marcadores) regularmente.

    Exemplo:

      <ul>
      <li>Item 1</li >
      <li>Item 2</li >
      <li>Item 3</li >
      </ul>

    A lista aparecerá da seguinte forma no navegador:

    • Item 1
    • Item 2
    • Item 3

    Textos

    Um elemento chamado link é usado em HTML para estabelecer uma conexão entre várias páginas da Web ou recursos relacionados. Ele permite que os usuários cliquem em links para abrir documentos, navegar para outras páginas ou realizar determinadas ações.

    O link pode ser criado quando usamos a tag <a>

    (anchor), que significa "ancoragem", ou seja um direcionamento para outro lugar. Junto ao atributo "href" para ser possível especificar o destino do link, podendo ser um endereço de página web, um arquivo local ou um identificador interno da página.

    Exemplo:

    "<a href="https://www.torradinhatostada.com">Visite o site exemplo</a>"

    Quando um link é renderizado em um navegador, ele aparece como um texto sublinhado e, quando clicado, direciona o navegador para a URL especificada no atributo href. Além disso, você pode adicionar atributos opcionais ao link, como target para especificar em qual janela ou guia ele deve ser aberto (por exemplo, target="_blank" para abrir um novo guia) ou classes e estilos para personalizar a aparência do link usando CSS.
